# README

This folder is full of [Eza](https://github.com/eza-community/eza) themes.

Eza themes can be found at: https://github.com/eza-community/eza-themes

## Setup

I have in .zshrc:

```
export EZA_CONFIG_DIR="$HOME/.config/eza"
```

It defaults to only use `theme.yml`. The other files are there to copy/paste into theme.yml.
